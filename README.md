# HML Chemical Enterprises Website

A simple web application for chemical billing and inventory management.

## Features

- Dashboard with main sections for billing, item management, stock, sales, customers, suppliers, reports, user roles, extra features, and settings.
- Shop name: HML Chemical Enterprises
- GST Number, Address, Logo upload, etc.

## How to Run

Open `index.html` in a web browser.

## Future Enhancements

- Backend integration for database, SMS, PDF generation, etc.
- User authentication and role-based access.
- Mobile responsiveness.
- Multi-language support (English + Tamil).
- Dark mode toggle.
